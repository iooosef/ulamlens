# UlamLens

UlamLens is a Python application using PyQt for the GUI. This project is currently under development.

## Features
- PyQt-based GUI
- Modular Python code structure

## Getting Started

### Prerequisites
- Python 3.8+
- PyQt5 (or PyQt6)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/iooosef/ulamlens.git
   cd ulamlens
   ```
2. (Optional) Create and activate a virtual environment:
   ```sh
   python -m venv venv
   .\venv\Scripts\activate
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

### Running the Application
```sh
python main.py
```
